**Announcing the** [**Unicode Emoji character map**](https://toolstud.io/web/emoji.php) **on toolstud.io**! It allows you to search for Emojis, like on the word &#8216;fruit&#8217;, and see all the Emojis that use it in their description. If you select 1 Emoji, you get more detailed information on it, and all the available encodings, ready to copy/paste.<figure class="wp-block-image">
